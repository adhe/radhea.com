# radhea.com

Personal portfolio site for Adhe — a product designer building digital products with Roundrive Digital.

Built with SvelteKit, Tailwind CSS, and mdsx. Case studies are written in Markdown and rendered as individual pages.

## Develop

```sh
bun install
bun run dev
```

## Build

```sh
bun run build
```

Configured for Cloudflare Pages via `@sveltejs/adapter-cloudflare`.
